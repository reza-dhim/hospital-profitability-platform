# Design System

## Design Direction
Tampilan harus terasa:
- enterprise
- modern
- clean
- profesional
- premium
- user friendly
- tidak terlihat seperti hasil vibe coding

Referensi rasa visual:
- Stripe Dashboard
- Linear
- Vercel
- Notion
- Microsoft Fabric
- SAP Fiori modern

## UI Principles
- Banyak whitespace
- Card-based layout
- Soft shadow
- Rounded corners
- Clear hierarchy
- Sidebar rapi
- Data table nyaman dibaca
- Chart tidak berlebihan
- CTA jelas
- Empty state informatif

## Mandatory States
Setiap halaman wajib punya:
- loading skeleton
- empty state
- error state
- success toast
- first-time guidance
- tooltip

## Onboarding UX
Saat user pertama login:
1. Welcome screen
2. Setup hospital profile
3. Setup cost center
4. Setup profit center
5. Download template
6. Upload sample data
7. Run calculation
8. View dashboard
