# UX Onboarding Guide

## First Login Experience
Saat user pertama login, jangan langsung tampil dashboard kosong.

Tampilkan onboarding wizard:

1. Welcome
2. Hospital Profile
3. Cost Center Setup
4. Profit Center Setup
5. Driver Setup
6. Download Excel Template
7. Upload Demo / Real Data
8. Validation Result
9. Run Calculation
10. View Executive Dashboard

## Product Tour
Gunakan spotlight overlay untuk menjelaskan:
- sidebar
- dashboard KPI
- upload center

- allocation engine
- AI insight
- report export

## Empty State Examples

### No Cost Center
Title: Belum ada Cost Center  
Description: Tambahkan unit non-profit seperti HRD, Laundry, IT, CSSD, dan lainnya.  
CTA: Tambah Cost Center / Import Template

### No Upload
Title: Belum ada data biaya  
Description: Upload data biaya menggunakan template standar agar sistem dapat menghitung costing.  
CTA: Download Template

### No Calculation
Title: Perhitungan belum dijalankan  
Description: Jalankan Cost Allocation untuk melihat unit cost, profit, dan margin.  
CTA: Run Calculation

## Tooltip Examples
Cost Allocation:
Distribusi biaya dari unit non-profit ke unit penghasil pendapatan berdasarkan driver tertentu.

Driver:
Parameter pembagi biaya seperti jumlah pegawai, volume cucian, luas area, hari rawat, atau jumlah tindakan.

Unit Cost:
Biaya aktual untuk menghasilkan satu unit layanan.

Margin:
Persentase profit dibanding revenue.
